BAJILLION WEB INTERFACE
=======================

1. Open this Assignment6 folder in VS Code.
2. Open a terminal in this same folder.
3. Run:

   python extension_server.py

4. Keep the terminal open.
5. Open a browser and go to:

   http://localhost:8000

6. Type a search such as:

   nice
   stanford
   cheap apple products

7. Stop the server with Ctrl+C in the terminal.
